# voke-buddhism-net

藏传佛教典籍结构化知识工程项目。

## 项目宗旨

将藏传佛教汉译典籍系统性地转化为结构化知识图谱（实体、事件、关系、传承、本体），并基于此产出面向公众的可视化探索界面。

首期源典：第二世敦珠法王《藏密佛教史》（索达吉堪布译）。

## 方法论来源

本项目的管线骨架与方法论直接借鉴**西瓜（鲍捷）开源的 [shiji-kb](https://github.com/baojie/shiji-kb) 项目**——一个用 AI 构造《史记》知识库的工程实践，包含 14 个 META 技能 + 跨 10 阶段的近百份可执行 SKILL 文档。

向 shiji-kb 项目和西瓜致谢。本项目复用其方法论，针对藏传佛教典籍的特殊性（多叙述层、师承体系、转世结构、多历法时间）做了改造。
