# 叙述层标签规范 (Narrative Layers)

> **状态**：v0.2 — **已审核（基线）**
> **作用范围**：本项目所有从源典抽取的事实断言（实体属性、事件、关系三元组）
> **数据来源**：本规范的所有例句均取自敦珠《藏密佛教史》第三品 · 藏传佛法

---

## 一、为什么需要叙述层

藏传佛教典籍里，同一个"事实"经常出自完全不同的"证据来源"，而这些来源的**认知地位天差地别**。最经典的例子来自第三品：

> 莲师到底在吐蕃住了多长时间呢？**短时间的观点**认为莲师在藏地住了六个月或十八个月；**中等时间的观点**认为住了三年或六年或十二年；**长时间的观点**将每两季算为一年，共住了一百一十一年，也就是五十五个整年零六个月。……自宗的诸位智者上师们的说法与这些史实都无有矛盾之处。

如果我们把"莲师驻藏 = 6 个月"和"莲师驻藏 = 111 年"都当作平等的"事实三元组"扔进知识图谱，下游的所有时间推理就全都废了。这两条断言的"硬度"完全不同：

- "6 个月"出自《巴协》—— **典据**
- "111 年"出自"长时间观点"—— **传闻**
- "55.5 年"出自"自宗诸位智者"—— **宗派(宁玛)**
- "莲师可同时示现于多刹土"—— **神变**

**叙述层标签**就是给每条断言打上"它的证据是什么类型"的元数据。它不消除分歧——它把分歧分层、可追溯、可分别推理。

## 二、核心原则

| # | 原则 |
|---|---|
| 1 | **每条事实断言都必须有至少一个叙述层标签**。无标签 = 缺数据，不允许进入 KG |
| 2 | **一条断言可以同时带多个层**（multi-label），例如典据 + 授记、宗派 + 神变 |
| 3 | **史传 与 神变 互斥**：一条断言要么属于凡常人认知可能发生的范畴（史传），要么必须诉诸密法本体论才成立（神变）。两者不能同时打 |
| 4 | **不强行消解多源分歧**——多个相互矛盾的断言可以并存，各带自己的层标签 |
| 5 | **层标签与"实体/事件/关系"类型正交**——是元数据，不是数据本身 |
| 6 | **层标签 与 作者态度 正交**——层描述证据类型（中性），作者立场用单独的 sub-tag 表达 |
| 7 | **每条断言带置信度** `confidence ∈ [0.0, 1.0]`——下游推理可按置信度加权 |
| 8 | **下游推理必须按层做差异化处理**——史传层断言可参与时间约束推理，神变层断言不可 |

## 三、七个叙述层（v0.2）

### 1. 史传层 `史传`

**定义**：作者或传统以"曾发生"的口吻直接陈述、且**符合凡常人认知**的历史性叙事。这是默认层——一段没有任何标记词、内容上也无超凡因素的陈述自动属于史传层。

**关键边界**：
- 只要事件本身在凡常自然法则内（出生、即位、迎娶、建寺、出家、翻译、教学、圆寂等）→ 史传
- 一旦事件涉及超凡因素（化身、神通、降伏鬼神、多刹土同时示现、空中飞行、改变地形等）→ **神变**，不再是史传
- 主体是不是上师/本尊不影响判定——决定层的是**事件本身**，不是事件的**主体**
- "莲师从黑波山步行到桑耶寺" = 史传（步行是凡常）
- "莲师腾空而飞" = 神变（飞行是超凡）

**文本特征**：
- 没有"据说""按某经""授记"等前导词的陈述句
- 包含具体人物、地点、年代、行动的叙事
- 多用"是""为""已""了"等完成态
- 内容无超凡元素

**第三品真实例子**：
- 「国王赤松德赞在十三岁时执掌国政，二十岁发心弘扬正法。」
- 「巴赤则出家为僧并受了比丘戒，取名为花扬，国王也赞称他为巴绕那。」
- 「在建造大昭寺时……国王与二位王妃闭关修行本尊。」（建造与闭关都是凡常行为）
- 「鲁墨大师培养出四顶梁之柱的弟子、八大栋梁弟子、三十二椽木弟子、千数短木弟子等等。」
- 「莲师又焚烧了损害佛法、势力强大的外道，建起了千万佛塔。」（焚烧建塔本身凡常；但若同章节强调"幻化"则改归神变）

**下游用途**：
- SKILL_04（事件构建）：可以正常抽取事件
- SKILL_05（关系构建）：可以正常生成 SPO 三元组
- SKILL_07（逻辑推理）：可以参与时间约束、因果链、人物生卒年推断

---

### 2. 典据引用层 `典据`

**定义**：明确归属于某部经/论/史/续的引用。作者借助权威经典来支持一个断言。

**文本特征**：
- 「《X》中说……」「《X》云……」「《X》中所说的……」
- 「按照《X》中……」「依据《X》……」
- 「这是在《X》中明显记载的」
- 引文通常用引号包围

**第三品真实例子**：
- 「按照《文殊根本续》中"雪域由湖减，出现娑罗林"的授记」（同时是授记层）
- 「《无垢天女授记经》中授记说："我入灭后两千五百年佛法将弘扬于红面人之境内。"」
- 「这是在《巴协》中明显记载的」
- 「《文殊根本续》中说："名为具天境，雪域之境内，住有人君王，出生乐匝族……"」
- 「按照《巴协》等中"只是住了短暂的时间"来讲」
- 「这也就像《贤愚经》、《毗奈耶经》中所说的佛于舍卫城的神变天数显得有多有少一样」

**强制属性**：
- `source_book`：出处书名（必填）
- `quote`：引文原文（如有，必填）

**下游用途**：
- 典据 ↔ 主张构成 `cites` 关系
- 同一断言被多部典籍并引时，置信度递增

---

### 3. 授记层 `授记`

**定义**：来自佛经/续部/祖师授记中的预言性陈述。授记可能"已应验"或"未应验"，但其性质永远是"预言"，而非历史观察。

**文本特征**：
- 出现"授记"二字
- 「《X》中授记……」「按照……授记」
- 内容通常是"未来某时将会……"
- 也可以是已应验的授记（"我入灭后两千五百年佛法将弘扬于红面人之境内"——回看时已应验）

**第三品真实例子**：
- 「《无垢天女授记经》中授记说："我入灭后两千五百年佛法将弘扬于红面人之境内。"」
- 「按照经中授记，第二十八代王朝，普贤菩萨的化身拉托托日涅赞……」
- 「当时他得到'再过五个朝代以后会有人对经中含义心领神会'的授记」
- 「祖父松赞干布曾亲自授记'从现在起到第五代时，我的曾孙国王德时期出现正法。'」
- 「莲师……对国王赐予了许多未来的授记」
- 「莲师又隐藏了一百国王魂伏藏……并一一授记开取的时间、取伏藏的人、有缘的法主等。」

**下游用途**：
- 授记层断言**不进入历史时间线**，但可单独构成"授记网络"
- SKILL_07（逻辑推理）：授记应验/未应验状态可作为单独的判断对象
- 伏藏事件中的"开取授记"是 terma chain 推理的前置链
- 授记层与典据层经常合并出现（"《X》中授记"），多标签即可

---

### 4. 宗派层 `宗派(派别名)`

**定义**：明确归属于某个具体教派/传承的立场或解释。**层标签必须带派别参数**。这是 v0.1 中"自宗层"的扩展——既包括作者本宗的立场，也包括其他宗派的立场。

**派别参数（初始集合，可扩展）**：
- `宁玛`（敦珠书的本宗）
- `噶举`
- `萨迦`
- `格鲁`
- `觉囊`
- `希解`
- `觉域`
- `时轮`（严格说是教法派系，非独立宗派；视使用而定）
- `苯`（苯教）
- `具名祖师` ——形如 `宗派(讷巴班智达)`，当只能定位到具名祖师而不能定位到稳定教派时使用
- 派别内部可能还有分歧，未来可加二级 sub-tag（如 `宗派(宁玛/龙钦巴系)`）

**文本特征**：
- 「自宗诸位智者上师们……」（本宗立场，敦珠语境下 = 宗派(宁玛)）
- 「许多智者认为……」（具名度低时归 宗派 还是 传闻 须看上下文）
- 「X 派认为……」「X 教徒说……」（明确他派立场）
- 「讷巴班智达认为……」（具名祖师，归 `宗派(讷巴班智达)` 或其所属派别）

**第三品真实例子**：
- 「自宗的诸位智者上师们的说法与这些史实都无有矛盾之处」 → `宗派(宁玛)`
- 「许多智者认为：大师作这样的顿圆传戒……是没有过错的」 → `宗派(宁玛)`（在敦珠书的上下文中）
- 「讷巴班智达认为'从天而降'的说法是因为苯波教喜欢天空，所以纯属苯波传言」 → `宗派(讷巴班智达)`，作者态度 = 认可
- 「苯波教徒们说'从登天绳降下，该是天人'」 → `宗派(苯)`，作者态度 = 怀疑或反驳

**强制属性**：
- `sect`：派别参数（必填）
- `asserter`：具名提出者（如有，可选）

**下游用途**：
- 跨派别比较——同一议题不同派别立场对照
- 派别内分歧也可记录（同 `sect` 不同 `asserter`）
- 应用层可生成"派别立场对照视图"

### 5. 神变层 `神变`

**定义**：必须诉诸佛教/密法本体论才能成立的陈述。包括但不限于：身体多刹土同时示现、神通改变自然法则、化身在不同时空同时活动、本尊化身宣称、降伏鬼神、伏藏埋藏、瞬移、空中行、改变地形等。这类陈述在凡常历史框架下不可解，需要在"法身-报身-化身"的本体论中才有意义。

**与 史传 互斥**：一条断言不能同时打 `史传` + `神变`。判定准则：**事件本身是否符合凡常自然法则**。

**包含的子情形**：
- ✅ 身相变化（示现忿怒/寂静相、化现千身）
- ✅ 神通飞行（腾空、瞬移、神足通）
- ✅ 降伏非人（鬼神、龙、罗刹、空行）
- ✅ 多刹土同时示现
- ✅ 改变物理（让山川闭合、让沙漠出水、让岩石生印）
- ✅ **本尊化身宣称**（"X 是观音的化身"——这是形而上学陈述，不是历史观察）
- ✅ 伏藏埋藏与未来掘取
- ✅ 死后转世链条上的同一性宣称

**文本特征**：
- 出现"示现""幻化""化现""神变""神通""加持""降伏""非人""鬼神"等关键词
- 主体是"真身/化身/二种身相"
- 行动跨越自然法则（瞬移、空中行、改变地形、降伏鬼神）
- 时间/空间不连续
- "X 是 Y 的化身/示现"句式

**第三品真实例子**：
- 「莲师具有于一切刹土中同时示现身相等不可思议的神变」
- 「莲师示现令人望而生畏的忿怒威猛相，降伏了藏地大大小小的所有凶神恶煞」
- 「莲师示现第二种身相，从洞巴戒创之外印藏交界的山顶离去，在人们的眼前腾空而飞」
- 「真身一直安住在耶多卓、青普等雪山静处，为有缘分的君臣转无上密的法轮」
- 「这时藏地的鬼神们降下大雪大雨，并使所有山川相互闭合，莲师则从山间畅通无阻穿行而过」
- 「莲师把雪山湖泊全部加持成修行地」
- 「整个工程白天由人工建造，晚上非人建得更高」（桑耶寺兴建）
- **「普贤菩萨的化身拉托托日涅赞」**（化身宣称 → 神变）
- **「观世音菩萨示现为人君的形象法王松赞干布」**（化身宣称 → 神变）
- **「文殊菩萨幻化人君赤松德赞」**（化身宣称 → 神变）
- **「文殊菩萨的化身囤弥桑布札」**（化身宣称 → 神变）
- **「金刚手菩萨的化身——赤热巴坚」**（化身宣称 → 神变）
- 「莲师又隐藏了……不可胜数的伏藏品，并一一授记开取的时间、取伏藏的人、有缘的法主等」（伏藏埋藏 → 神变 + 授记）
- 「莲师从札玛湖以前是一片沙滩，全部变成了草原」（改变地形 → 神变）

**下游用途**：
- 神变层事件**可以进入事件库**，但带 `非凡常时空` 元数据
- SKILL_07：神变层断言**不参与**人物生卒年/位置一致性等时间空间约束推理
- 神变层是理解"莲师驻藏 ≈ 永恒"这类陈述的关键
- 应用层（SKILL_09）：神变事件可以作为视觉化的"神话叙事线"独立呈现

### 6. 证境层 `证境`

**定义**：来自非凡常感知状态的认知——梦境、净相、禅定境界。区别于 `神变`（外在的超凡事件），证境层捕捉的是**内在的非凡常感知**——上师在梦中得授记、在净相中见本尊、在禅定中得教法等。这是宁玛派窍诀传承与伏藏系统的核心来源类型之一。

**Sub-tag**（必填一个，描述具体子类型）：
- `梦` ——睡眠中的梦境
- `净相` ——清醒但处于清净观见的状态（dag snang）
- `禅定` ——三摩地中所见所闻所证

**与 神变 的区别**：
- 神变 = 外在的超凡事件（旁观者也能"看到"）
- 证境 = 个体内在的非凡常感知（旁观者通常看不到，是当事人独享的体验）
- 边界案例："X 在禅定中见到本尊授记" → `证境(禅定)`，因为是 X 个人的体验
- "莲师当众腾空而飞" → `神变`，因为是公开的外在事件

**文本特征**：
- 出现"梦""梦中""梦见"
- 出现"净相""清净观见""现见"
- 出现"定中""禅定中""三摩地"
- 通常关联到"得授记""得教法""得灌顶""见本尊"

**第三品真实例子**（章节 3 例子较少，证境层的主战场是第五品远传经幻心、第六品近传伏藏史；这里仅给现有最接近的）：
- 「请一些诸根敏锐、聪明伶俐的小孩看圆光，结果他们说出了……藏地残暴天神及龙类的名称、行为、住处」 → `证境(净相)`，圆光是清净观见的一种
- *(更典型的梦/净相/禅定例子留待标注第五、第六品时充实)*

**强制属性**：
- `sub_type`：梦/净相/禅定 之一（必填）
- `experiencer`：体验者（通常是某位上师，必填）

**下游用途**：
- 证境层断言进入事件库，但与 神变 类似，**不参与凡常时空推理**
- 证境层与"教法源流"链条紧密相连——某个窍诀的"证境得法"事件是其传承的起点
- SKILL_05 关系构建：证境事件 → `received_by_vision` 关系，连接"上师"与"教法/经函/伏藏"

---

### 7. 传闻层 `传闻`

**定义**：作者明确标记为"听来的""无确凿出处的""无具名归属的"陈述。它是一个**中性**的来源类型——并不必然意味着作者不信，只意味着证据链不强。

**与 v0.1 的区别**：
- v0.1 名为"传言"，含贬义。v0.2 改名"传闻"，强调中性
- v0.1 把"讷巴班智达认为"和"据说"都归在这里。v0.2 中具名祖师的判断归 `宗派(具名祖师)` 或所属派别，纯无具名的归 `传闻`

**文本特征**：
- 「据说……」「据传……」
- 「有些人认为……」（无具名时）
- 「人们一致共称……」「共称为……」（弱化的口耳传统）
- 没有典据引用、没有具名祖师、没有派别归属的弱化陈述

**第三品真实例子**：
- 「据说在预试七人还没有出家之前，尼占帕翁卡地方也出现了一百位左右修行观音法、头挽发髻的在家瑜伽士。」
- 「人们一致共称：首先在湖泊逐渐减少形成郁郁葱葱的森林时期，普陀山大悲观音加持的一只猴子来到雪域，他和岩罗刹女交配，由此繁衍藏人。」（口耳传统）
- 「有些认为这位国王是百军王的太子。」（无具名）

**下游用途**：
- 传闻层断言**记录但默认不参与正面推理**（置信度通常较低）
- 它对"反驳/异说网络"的构造有用——可以追溯本派对哪些他派说法做了批驳
- SKILL_07e（真实性推理）的主要素材
- 不要把传闻层等同于"虚假"——很多伏藏师传记的早期口耳传统也归此层

## 四、正交维度

层标签捕捉"证据来源"，但每条断言还有两个**正交维度**需要标记：

### 4.1 作者态度 sub-tag `author_stance`

层标签是中性的——它只描述证据类型。作者对这条断言的立场要单独打：

| 值 | 含义 | 例 |
|---|---|---|
| `认可` | 作者主动陈述、明显接受 | "自宗的诸位智者上师们的说法与这些史实都无有矛盾之处" → `宗派(宁玛)` + 认可 |
| `中立` | 作者只是记录、不表态 | "据说在预试七人还没有出家之前……" → `传闻` + 中立 |
| `怀疑` | 作者表达疑虑或不完全采信 | "尽管有各种各样的说法……" → 视上下文 + 怀疑 |
| `反驳` | 作者明确否定 | "讷巴班智达认为'从天而降'的说法是因为苯波教喜欢天空，所以纯属苯波传言" → 内嵌的"从天而降"被作者通过讷巴班智达 反驳 |

**判定原则**：默认 `中立`。出现"事实上""实际上""并非如此""纯属""恶劣"等强标记词时升级为"认可/反驳"。

### 4.2 置信度 `confidence`

每条断言带一个 `confidence ∈ [0.0, 1.0]` 数字。它表示**对该断言在其层内的可信度的标注者评估**——不是"该断言客观为真的概率"。

**初始默认值（标注者可按上下文 override）**：

| 层 | 默认 confidence | 理由 |
|---|---|---|
| `典据` | 0.95 | 有明确出处书名 + 引文 |
| `授记` | 0.95 | 文本明确记录 |
| `史传` | 0.85 | 高但非绝对，史书也会错 |
| `宗派` | 0.85 | 派别官方立场 |
| `神变` | 0.90 | 文本忠实度高（不评判形而上真伪） |
| `证境` | 0.85 | 个人体验记录 |
| `传闻` | 0.50 | 弱证据 |

**特殊情况**：
- 同一断言被多种层并列支持 → 总置信度按最高层取
- 同一断言被多源引用 → 置信度可在 SKILL_07 阶段额外加权
- 标注者主观下调时必须在 `note` 字段说明理由

## 五、多层组合（multi-label）

一条断言**经常**同时属于多层（除 `史传` ↔ `神变` 互斥外）。下面是第三品里的真实组合例子：

| 文本 | 标签组合 | 解释 |
|---|---|---|
| 「《无垢天女授记经》中授记说："我入灭后两千五百年佛法将弘扬于红面人之境内"」 | `典据 + 授记` | 出自经典（典据），且内容是预言（授记） |
| 「按照《巴协》等中"只是住了短暂的时间"来讲」 | `典据 + 史传` | 引用《巴协》（典据），陈述的内容是凡常历史（史传） |
| 「莲师又隐藏了……不可胜数的伏藏品，并一一授记开取的时间……」 | `神变 + 授记` | 伏藏埋藏是神变；授记开取时间是授记。**不打史传**——伏藏埋藏不是凡常事件 |
| 「自宗的诸位智者上师们的说法与这些史实都无有矛盾之处」 | `宗派(宁玛)`，态度=认可 | 仅本派立场陈述 |
| 「莲师示现第二种身相，从洞巴戒创之外印藏交界的山顶离去，在人们的眼前腾空而飞」 | `神变` | 整个事件是神变；**不打史传**——示现+腾空都是超凡 |
| 「讷巴班智达认为'从天而降'的说法……纯属苯波传言」 | 主断言：`宗派(讷巴班智达)`+认可<br>嵌套断言："从天而降" → `宗派(苯)`+反驳 | 嵌套：本派祖师反驳他派说法 |
| 「《文殊根本续》中说："名为具天境……" 依据这一授记，拉托托日以后的第五代，观世音菩萨示现为人君的形象法王松赞干布」 | 前半：`典据+授记`<br>后半：`神变`（化身宣称） | 一句话两个断言，分别打标 |
| 「按照经中授记，第二十八代王朝，普贤菩萨的化身拉托托日涅赞居于皇宫永布拉冈的时候，一个宝箧落到王宫顶上」 | 化身宣称：`典据+授记+神变`<br>宝箧落顶：`神变` | 两个断言分别打标 |

**判定原则**：
- **能加就加**——多标签不会浪费，单标签可能丢信息
- **非互斥的层都可以叠加**——典据/授记/宗派/神变/证境/传闻 之间均可叠加
- **史传 ↔ 神变 互斥**——这是 v0.2 引入的硬约束
- **嵌套用反驳关系单独表达**——一个断言反驳另一个断言时，两条断言独立标层，反驳关系单独建模

## 六、常见叙述模式

### 6.1 异说并存模式

作者并列多种说法，**不挑选**正确答案。第三品最经典的例子是莲师驻藏时长的六说并存。

**处理方式**：每种说法独立成一条断言，各自打层。它们之间的"并列"关系可以由 SKILL_05 单独建模为 `alternativeOf` 三元组。

### 6.2 反驳传闻模式

`宗派(具名祖师)` 反驳 `传闻` 或 `宗派(他派)` 的说法。第三品例子：讷巴班智达反驳"从天而降"的苯教说法。

**处理方式**：原说法 + 反驳说法 各自独立断言，二者之间建 `refutes` 关系。

### 6.3 调和史料模式

`宗派(宁玛)` 的智者用"两季为一年"等解释来调和不同史料的数字差异。第三品例子：长时间观点的 111 年实际是 55.5 年。

**处理方式**：原断言保留，调和断言新建一条，二者之间建 `reconciles` 关系。

### 6.4 神变兜底模式

`神变` 层断言被用来"兜底"史传层的矛盾——既然莲师可以多刹土同时示现，那么短期/长期都可以同时为真。

**处理方式**：神变兜底属于 `宗派(宁玛) + 神变` 双层断言，且与被调和的史传断言之间建 `dharmakaya-reconciles` 关系。

## 七、暂定标注语法（仅作 Sprint 1 占位，最终由 SKILL_03a/04a 定稿）

```yaml
# 单层
fact: 莲师驻藏 6 个月
layers: [史传]
source_book: 《巴协》
sub_layer: 短时间观点
author_stance: 中立
confidence: 0.85

# 多层 + 派别
fact: 普贤菩萨化身为拉托托日涅赞
layers: [典据, 授记, 神变]
cites: 《无垢天女授记经》
relation_type: manifestationOf
author_stance: 认可
confidence: 0.95

# 宗派 + 反驳
fact: "从天而降"的说法纯属苯波传言
layers: [宗派]
sect: 讷巴班智达
author_stance: 认可
confidence: 0.90
refutes: [fact_id_of_从天而降]

# 神变兜底
fact: 莲师驻藏 55.5 年
layers: [宗派, 神变]
sect: 宁玛
asserter: 自宗诸位智者上师
author_stance: 认可
confidence: 0.85
reconciles: [莲师驻藏 111 年, 莲师驻藏 6 个月]

# 证境
fact: 圆光中所见鬼神名称行为住处
layers: [证境]
sub_type: 净相
experiencer: 看圆光的孩童
author_stance: 中立
confidence: 0.80
```
